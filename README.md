# QRcode Generator

this is qrcode generator using [qrcode](https://www.npmjs.com/package/qrcode)

```
git clone https://github.com/Ryan-Willpower/qrcode-generator.git
npm install
npm start
```

or if you use yarn

```
git clone https://github.com/Ryan-Willpower/qrcode-generator.git
yarn install
yarn start
```
